# NAVAJA-SUIZA
NAVAJA
Esta es mi Navaja suiza personal para trabajadores de Administradores de Sistemas. Tiene diferentes herramientas basicas para facilitar el trabajo diario.
Para poder utilizar los script en powersheell debemos activar primero su uso de ahi el archivo 'activar uso de script'. Una vez hemos realizado nuestras laboras, es oportuno volver a dejar estops parametros como estaban por seguridad, de ahi la imagen "desactivar uso de scrips"
Para descargar el script y poder ejecutarlo sin que Windows lo bloquee debemos descargarnos el archivo .bat "NAVAJA-SUIZA-DESCARGA.bat" y ejecutarlo para que haga todo automáticamente: descargar el script, desbloquear el archivo y ejecutarlo, sin que el usuario tenga que tocar nada de PowerShell.
